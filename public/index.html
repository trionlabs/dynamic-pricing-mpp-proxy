Dynamic Surge Pricing

A 5-tier pricing engine in a Durable Object tracks demand per route with a sliding window and EMA smoothing. Prices rise automatically under load.

Workers AI Gateway

No external API keys needed. The Worker runs AI inference directly via the Workers AI binding — Cloudflare's built-in LLM platform.

MPP + Tempo

HTTP 402 challenges with WWW-Authenticate: Payment. AI agents with wallets pay per-request via Tempo. JWT cookies provide 1-hour repeat access.

How it works

01
Agent requests AI completion

POST /api/chat with messages. Worker returns 402 with payment challenge and current surge price.

02
Agent pays via MPP

The agent's wallet signs a Tempo payment for the dynamic price. Retries with Authorization: Payment.

03
Worker runs inference & returns receipt

Workers AI runs the model, response includes Payment-Receipt and a JWT cookie for repeat access.

04
Price adjusts with demand

Each request is recorded. More agents = higher price. The Durable Object smooths pricing to prevent whiplash.
